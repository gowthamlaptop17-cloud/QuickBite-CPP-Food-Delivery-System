# QuickBite - Food Delivery System

A console-based Food Delivery System built in C++17 demonstrating Object-Oriented Programming principles and software design patterns.

## Features

- Customer Registration & Login
- Restaurant Management
- Menu Browsing
- Order Placement
- Order Tracking
- Payment Processing
  - UPI
  - Card
  - Cash on Delivery
- Customer Profile Management

## OOP Concepts Used

- Abstraction
- Encapsulation
- Inheritance
- Polymorphism
- Singleton Design Pattern
- Smart Pointers

## Technologies

- C++17
- STL
- Object-Oriented Design
